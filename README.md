# JustKeep

### The pyrpose:
Make a simple To do list on Python
### Languages:
- Python 3.12
### System requirements:
- Linux | Mac OS | Windows
### Quick setup
1. Clone project
2. Upgrate pip:
`pip install --upgrade pip`
3. Install module Pillow (PIL)
`pip install Pillow`
4. Run file `justkeep.py`
